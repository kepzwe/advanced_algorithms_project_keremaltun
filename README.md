# Pathfinding Optimization - Potsdam
This project implements **Dijkstra** and **A*** algorithms to find the shortest path to the nearest hospital in Potsdam, Germany.

## Project Overview
- **City:** Potsdam, Germany
- **Algorithms:** Dijkstra's Algorithm & A* Search
- **Goal:** Optimize emergency response times by identifying the most efficient routes to 4 major hospitals.

## Scenarios Tested
1. Museum Barberini -> Ernst von Bergmann Hospital
2. Brandenburger Tor -> St. Josefs-Krankenhaus
3. Park Babelsberg -> Oberlinklinik
4. Einstein Tower -> Ernst von Bergmann Hospital
5. New Palace -> EvB Klinikum West
